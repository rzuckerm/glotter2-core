=============
Glotter2-Core
=============

Welcome to the glotter2-core documentation!

This package is a simple, lightweight core library for
`Glotter2 <https://github.com/rzuckerm/glotter2>`_.
It can be used instead of Glotter2 when all that is needed is settings,
project information, and test information.
